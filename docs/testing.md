# Testing

This document describes the testing strategy for the application.

The project includes the following types of tests:

* **Unit tests:** These tests are located in the `app/src/test` directory and are run on the local JVM.
* **Android tests:** These tests are located in the `app/src/androidTest` directory and are run on an Android device or emulator.

The project uses the following testing frameworks:

* **JUnit:** A popular testing framework for Java and Kotlin.
* **Espresso:** A testing framework for Android UI tests.
* **Compose Test:** A testing framework for Jetpack Compose.
