# File Descriptions

This document provides a description of the files and directories in the project.

## Root Directory

* `AGENTS.md`: Instructions for AI agents working on the codebase.
* `docs/`: Contains detailed documentation about the project.
* `app/`: The main application module.
* `build.gradle.kts`: The main build script for the project.
* `settings.gradle.kts`: The settings script for the project.

## `app` Module

* `src/main/kotlin/com/hereliesaz/nolawallet/`: The main source code for the application.
* `src/main/kotlin/com/hereliesaz/nolawallet/data/`: Contains the data layer of the application.
* `src/main/kotlin/com/hereliesaz/nolawallet/ui/`: Contains the UI layer of the application.
* `src/main/kotlin/com/hereliesaz/nolawallet/viewmodel/`: Contains the ViewModel layer of the application.
* `src/main/kotlin/com/hereliesaz/nolawallet/MainActivity.kt`: The main entry point for the application.
* `build.gradle.kts`: The build script for the `app` module.
