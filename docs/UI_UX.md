# UI/UX

This document describes the user interface and user experience of the application.

## Onboarding Screen

The onboarding screen is the first screen the user sees. It provides a brief introduction to the application and prompts the user to log in.

## PIN Screen

The PIN screen is used to protect access to the user's wallet. The user must enter a 4-digit PIN to unlock the wallet.

## Wallet Screen

The wallet screen displays the user's licenses. The user can tap on a license to view its details.

## License Detail Screen

The license detail screen displays the details of a license, including the user's photo, name, and other information. The user can share their license from this screen.

## Share License Screen

The share license screen allows the user to share their license with others.

## Secret Config Screen

The secret config screen is a hidden screen that can be accessed from the wallet screen. It allows the user to configure the application's "secret" settings.
