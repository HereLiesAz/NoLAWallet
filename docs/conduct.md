# Code of Conduct

This document outlines the code of conduct for contributors to this project.

We are committed to providing a friendly, safe and welcoming environment for all, regardless of gender, sexual orientation, disability, ethnicity, or religion.

This code of conduct outlines our expectations for participant behavior as well as the consequences for unacceptable behavior.

We invite all sponsors, collaborators, and contributors to help us create a safe and positive experience for everyone.
