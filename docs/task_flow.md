# Task Flow

This document describes the flow of tasks in the application.

1. **Onboarding:** The user is greeted with the onboarding screen.
2. **PIN Entry:** The user enters their PIN to unlock the wallet.
3. **Wallet:** The user can view their licenses.
4. **License Detail:** The user can view the details of a license.
5. **Share License:** The user can share their license.
6. **Secret Config:** The user can access the secret config screen from the wallet.
